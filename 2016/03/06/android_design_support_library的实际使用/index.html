Android Design Support Library的简单学习

Android Design Support Library的简单学习

该库提供了许多高级组件，便于开发者开发Material Design的应用。

目前最新的Android Studio新建项目已经自动依赖了该库
‘com.android.support:design:23.2.0’

1. TextInputLayout

Layout which wraps an EditText (or descendant) to show a floating label when the hint is hidden due to the user inputting text. Also supports showing an error via setErrorEnabled(boolean) and setError(CharSequence).

配合EditText一同使用

注意：里面还有个TextInputEditText，不可与TextInputLayout同时使用。

2. FloatingActionButton

该控件使用简单，注意它的浮动位置，间接继承自ImageButton

3. Snackbar

基本用法与Toast类似，更注重互动操作，在屏幕底部显示。

Snackbars can contain an action which is set via setAction(CharSequence, android.view.View.OnClickListener).To be notified when a snackbar has been shown or dismissed, you can provide a Snackbar.Callback via setCallback(Callback).

4. NavigationView

抽屉式导航栏菜单项目
嵌入DrawerLayout中使用

通过drawerLayout.closeDrawers()，来开关

一般根据是否点击Toolbar左上角来控制开关

5. TabLayout

一般搭配ViewPager一同使用

6. CoordinatorLayout

它是用来组织它的子views之间协作的一个父view。CoordinatorLayout默认情况下可理解是一个FrameLayout，它的布局方式默认是一层一层叠上去

可包裹 Toolbar 来制造折叠效果
